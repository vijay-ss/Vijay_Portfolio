Covid-19 Interactive Dashboard


June 11, 2020

With the onset of Covid-19, people across the globe have been isolated in their homes without any contact with the outside world. Coupled with the constant bombardment of frightening news, this is a strong recipe for mental health complications.
This dashboard is intended to provide people with an unbiased, objective view of the current state of cases, without the fear-mongering.

Some key data manipulations include:

  • Extensive data cleaning and reshaping using Python for usability within the app.
  • Scraped twitter to pull Covid related tweets, reshaping the data into a data frame.
  • Sentiment analysis dividing the tweets into positive, negative and neutral cases.
  • Built the customer-facing UI using Plotly, Dash (Flask), HTML, CSS and Python.
