Opening a Toronto Fitness Centre


March 20, 2020

With the popularity of fitness centres and gyms on the rise in recent years, a lucrative business opportunity exists for those seeking to open one of their own in Toronto, Canada. This research project examined the ideal location to open a fitness centre in Toronto, based on characteristics within local neighbourhoods. It was concluded that the downtown core is the best area, within the Financial District. It has almost no local competition and plenty of nearby venues to drive traffic.

Some key technical project highlights include:

  • Web scraped all Toronto neighbourhood data using the BeautifulSoup package.
  • Retrieved top local neighbourhood venues using the Foursquare Developer API.
  • Cleaned and transformed input data into one consolidated data set for analysis.
  • Performed k-Means Clustering machine learning algorithm to generate 5 key centroids based on neighbourhood location and top local venues.
